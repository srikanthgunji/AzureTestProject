Quick Deploy (CLI)

Use Azure Storage Static Website or Azure Static Web Apps. For Storage, the gist is:

  1. Create a storage account
  2. Enable Static website with index.html as the index
  3. Upload this file to the $web container
Show sample Azure CLI commands 
# 1) Create a resource group & storage account (adjust names/region)
az group create -n my-rg -l eastus
az storage account create -n mystaticdemo123 -g my-rg -l eastus --sku Standard_LRS

# 2) Enable static website hosting
az storage blob service-properties update \
  --account-name mystaticdemo123 \
  --static-website \
  --index-document index.html

# 3) Upload the page to the $web container
az storage blob upload \
  --account-name mystaticdemo123 \
  -c "$web" -f ./index.html -n index.html --overwrite

# 4) Get the public URL
az storage account show -n mystaticdemo123 -g my-rg \
  --query "primaryEndpoints.web" -o tsv
